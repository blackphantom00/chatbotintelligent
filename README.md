# chatbotintelligent
This project is an intelligent AI-powered chatbot designed to automate conversations and provide accurate responses to user queries. The chatbot leverages Natural Language Processing (NLP) techniques to understand user input and generate relevant answers in real time.
## Installation & Usage

### Prerequisites

Before running the application, ensure that the following requirements are installed on your system:

* Python 3.10 or higher
* pip (Python Package Manager)
* Git

### Clone the Repository

```bash
git clone https://github.com/your-username/your-repository.git
cd your-repository
```

### Install Dependencies

Create and activate a virtual environment (recommended):

```bash
python -m venv venv
```

**Windows**

```bash
venv\Scripts\activate
```

**Linux / macOS**

```bash
source venv/bin/activate
```

Install the required packages:

```bash
pip install -r requirements.txt
```

### Launch the Application

#### Linux / macOS

```bash
bash run.sh
```

#### Windows

```bash
python app.py
```

### Access the Application

Once the application is running, open your browser and navigate to:

```text
http://localhost:5000
```
